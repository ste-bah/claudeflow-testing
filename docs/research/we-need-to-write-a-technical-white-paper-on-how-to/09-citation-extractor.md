## Usage Guidelines for Writers

### In-Text Citation Examples

**Parenthetical Citation**:
> Security Hub now provides near real-time analytics and risk prioritization (Amazon Web Services, 2025a).

**Narrative Citation**:
> Amazon Web Services (2025a) announced that Security Hub now provides near real-time analytics.

**Multiple AWS Sources in Same Year**:
> Cross-region aggregation (Amazon Web Services, 2025b) and central configuration (Amazon Web Services, 2025c) enable enterprise-scale deployments.

**Using Citation Keys**:
> The December 2025 GA announcement [AWS-SH-2025-01] introduced significant architectural changes.

### Tier-Based Citation Priority

1. **Tier 1 (AWS Official)**: Use for all factual claims, architectural guidance, and feature descriptions
2. **Tier 2 (Validated)**: Use for implementation patterns, cost analysis, and industry context
3. **Tier 3 (Community)**: Use only for discovery or practitioner perspectives with explicit caveats

### Disambiguation for Multiple AWS Sources

When citing multiple AWS sources from the same year, use lowercase letters:
- (Amazon Web Services, 2025a) - Security Hub GA announcement
- (Amazon Web Services, 2025b) - Central configuration
- (Amazon Web Services, 2025c) - OCSF documentation
